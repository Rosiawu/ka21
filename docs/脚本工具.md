# 脚本工具

当前仓库仅保留下列常用维护脚本，便于校验精选配置、热区数据以及批量检查工具链接。

## 链接巡检

### check-urls.js  (`node scripts/check-urls.js`)
- 批量访问 `src/data/tools.json` 内的工具链接，输出可用性报告。
- 支持 `--check-only`（仅摘要）与 `--timeout=<ms>` 参数，结果会生成 `scripts/invalid-tools.json`（运行时产物）。

## 图标转换

### convert-icons.js  (`npm run convert-icons`)
- 将 `public/icons/raw` 中的资源统一拷贝/转换到前端使用的路径，保持命名一致性。

## 精选配置校验

### validate-featured.ts  (`npm run validate-featured`)
- 构建前自动执行，验证精选池 ID 是否存在、备用池容量是否足够等。

### validate-featured-overlap.ts  (`npx tsx scripts/validate-featured-overlap.ts`)
- 辅助排查精选工具在人气/新品等模块之间的重复露出，输出重叠明细（需手动执行）。

## 热区配置验证

### validate-hot-section.ts  (`npx tsx scripts/validate-hot-section.ts`)
- 校验热门板块 JSON 配置（分类、排序、兜底池等）是否满足上线要求。

## 历史脚本

批量分类、标签、数据迁移等一次性脚本已从仓库移除。如需参考，可在历史提交中查找对应文件或根据 README 中说明自行编写脚本。
