# 更新日志

本文档记录数字生命卡兹克-KA21工具导航平台的所有重要变更。

## [未发布]

### 新增
- 优化项目文档结构，将长篇README.md拆分为多个专题文档，提高可读性和维护性
- 创建了docs目录，包含功能列表、技术架构、数据结构等专题文档
- 新增文档《文档结构.md》，提供文档结构的完整说明和导览
- 将工作日志和项目状态报告从home目录迁移至docs目录，统一管理文档资源
- 教程数据解耦方案，将教程数据从工具数据中分离出来
- 新增`tutorials.json`文件，独立存储教程数据
- 添加配置开关，支持新旧两种数据源的平稳切换
- 提供数据迁移脚本，自动从工具数据中提取教程信息
- 新增教程数据解耦文档，详细说明设计和使用方法

### 改进
- 重构导航系统，移除顶部分类筛选栏，将功能集成到侧边栏
- 优化萌新教程卡片交互体验，整个卡片可点击直接跳转
- 改进搜索状态视觉反馈，添加加载指示器
- 统一布局设计，确保教程页面与首页保持一致的设计语言
- 提高页面布局空间利用率，优化工具卡片设计减小内边距
- 实现按分类分区展示工具，提高内容组织结构
- 优化移动端侧边栏交互，点击外部区域自动关闭
- 改进组件性能，使用React.memo减少不必要的重渲染
- 优化教程数据结构，增加双向关联关系
- 重构教程数据加载逻辑，提高代码可维护性
- 更新README.md，添加数据管理相关说明

### 修复
- 修复搜索X按钮问题，同时清除输入内容和查询参数
- 解决特殊字符编码问题，确保所有分类名称正确显示
- 修复输入框类型，解决双重清除按钮问题
- 修复教程图片404错误问题，使用内联SVG替代外部图片文件
- 修复教程排序逻辑，确保在不同数据源下表现一致

### 移除
- 删除冗余的教程图片目录`public/tutorial-images`
- 移除未使用的预留功能目录
- 删除旧版不再使用的JSON文件
- 移除未使用的图片占位符逻辑，简化代码复杂度

## [1.2.0] - 2024-03-XX

### 新增
- 多维度工具排序功能，支持推荐排序、最新排序和名称排序
- 分类内独立排序系统，允许在每个分类区块内单独排序
- 通用排序工具函数，统一所有排序逻辑
- 顶部导航栏滚动自动隐藏功能，提升内容显示空间
- 动态生成的教程分类SVG背景，基于分类自动分配不同颜色方案

## [1.1.0] - 2024-02-XX

### 新增
- 集中式搜索功能整合，将工具和教程搜索结果统一展示
- 搜索快捷键支持系统，支持Ctrl+K/Command+K快速聚焦搜索框
- 侧边栏与工具分类直接集成，简化分类导航
- 导航面包屑系统，提供清晰的层级路径显示
- 可折叠侧边栏导航系统，优化空间利用
- 统一布局响应式系统，适应不同设备尺寸

### 改进
- 优化工具详情页布局，与首页保持一致的设计语言
- 强化标签系统统一化，创建AccessibilityBadge和RecommendBadge组件
- 增强工具卡片交互体验，添加平滑过渡动画
- 改进搜索结果页面布局，教程优先展示于工具之前
- 优化教程数据自动同步系统，避免重复添加相同文章
- Next.js 14客户端钩子优化，解决客户端钩子水合问题
- 优化图标系统，支持SVG和PNG双格式图标

### 修复
- 解决移动端导航问题，改进响应式布局
- 修复链接处理问题，添加安全属性
- 修复类型系统问题，增强类型安全
- 解决数据验证和修复功能中的问题
