# 技术架构

本文档说明平台的技术架构、目录结构与关键模块，已与当前代码库保持一致。

## 技术栈

- Next.js 14（App Router）
- React 18 + TypeScript
- Tailwind CSS
- ESLint

## 页面与数据流

- 首页：展示热门推荐（精选+兜底池、换一换）、分类工具、教程轮播、统计概览。
- 搜索：URL 参数驱动检索与筛选；统一搜索页同时展示工具与教程并支持排序。
- 详情：工具详情页显示指南、群友点评与相关文章/教程（优先 `relatedTutorials`）。
- 教程：支持关键词/分类/难度/技能筛选与排序。教程数据来源受 `USE_NEW_TUTORIALS_SOURCE` 控制。
- 数据：
  - 工具：`src/data/tools.json`（支持 `icons`/`icon`、`displayOrder`、`isVisible`、`relatedTutorials`）。
  - 教程：`src/data/tutorials.json`（由 `src/data/tutorials.ts` 统一映射为前端使用结构）。
  - 精选：`src/data/featured.json` + `FeaturedToolsManager` 组装热门推荐并提供分页“换一换”。

## 分析与埋点

- Clarity：通过 `ClarityAnalytics` 组件初始化，另有 `/test-clarity` 页面用于连通性与状态检查。
- Umami：`UmamiAnalytics` 组件（Cloud 脚本）。
- （已移除）百度统计：此前通过 `BaiduAnalytics` 组件接入，现已删除。
- 热门推荐埋点：`lib/hot-analytics.ts` 提供“换一换/点击”事件封装。

## 开发约定（摘要）

- 简单可读、最少状态、数据驱动渲染。
- SOLID+DRY：抽取 Hook/Util/通用组件以去重。
- 状态就近管理，避免不必要的状态提升与跨层传递。
