# 🔥 热门推荐板块优化 - 产品决策确认清单

## 背景说明
当前热门推荐页面显示的都是精选工具，缺乏视觉差异化。我们需要实现精选和非精选工具的混合展示，让用户能够区分重点推荐和一般推荐。

## 📊 当前数据调研

### 工具总数统计
- **总工具数量**：110个
- **必入级别工具**：18个（`recommendLevel: "high"`）
- **中等级别工具**：37个（`recommendLevel: "medium"`）
- **普通级别工具**：55个（`recommendLevel: "low"`）

### 工具分类分布
- **AI创作助手相关**：image(20个)、design(8个)、writing(4个)、audio(8个)、video(6个)
- **效率办公工具相关**：office(5个)、agent(8个)、chat(8个)、utils(20个)
- **学习成长工具相关**：coding(3个)、dev-platform(2个)、podcast(3个)、misc(8个)

### 当前热门推荐配置
- **AI创作助手**：2个精选工具（napkin, ima）
- **效率办公工具**：2个精选工具（deepseek, tongyi-efficiency）
- **学习成长工具**：2个精选工具（cursor, coze）
- **总计**：6个精选工具，但页面显示9个工具（说明有3个来自backupPool）

### 关键发现
1. **必入工具充足**：18个必入工具足够支撑精选展示需求
2. **非精选工具丰富**：92个medium/low级别工具提供充足选择
3. **当前问题**：backupPool可能优先选择了必入工具，导致全部显示精选标签

## 🎯 业务影响分析

### 用户体验影响
- **当前问题**：所有工具都显示精选标签，用户无法区分重点推荐
- **预期改善**：清晰的视觉层次帮助用户快速识别核心推荐工具
- **用户价值**：提升决策效率，减少选择困难

### 业务价值
- **核心资产展示**：18个必入工具是平台的核心价值，需要合理展示
- **工具曝光机会**：为medium/low级别工具提供展示机会，增加发现率
- **品牌专业性**：精准的精选标签提升平台专业形象和用户信任

### 2. 工具选择逻辑

#### 首页展示逻辑：
```javascript
function getHomepageTools() {
  const result = {};
  
  ['ai_creation', 'efficiency_tools', 'learning_tools'].forEach(category => {
    // 每个分类选择3个工具：2个精选 + 1个非精选
    const featuredTools = selectFeaturedTools(category, 2); // 从high级别选择
    const nonFeaturedTools = selectNonFeaturedTools(category, 1); // 从medium级别优先选择
    
    result[category] = {
      tools: [...featuredTools, ...nonFeaturedTools],
      featured_count: 2,
      non_featured_count: 1
    };
  });
  
  return result;
}
```

#### "换一换"后的逻辑：
```javascript
function getRefreshedTools(sessionHistory = []) {
  const result = {};
  
  ['ai_creation', 'efficiency_tools', 'learning_tools'].forEach(category => {
    // 混合策略：给非精选更多机会
    const featuredTools = selectFeaturedTools(category, 1, sessionHistory); // 减少精选数量
    const nonFeaturedTools = selectNonFeaturedTools(category, 2, sessionHistory); // 增加非精选数量
    
    // 随机混合排列，不按精选/非精选分组
    const mixedTools = shuffleArray([...featuredTools, ...nonFeaturedTools]);
    
    result[category] = {
      tools: mixedTools,
      featured_count: 1,
      non_featured_count: 2,
      display_strategy: 'mixed'
    };
  });
  
  return result;
}
```

### 4. 会话管理

```javascript
// 会话内去重和轮换管理
class RecommendationSession {
  constructor() {
    this.sessionId = generateSessionId();
    this.shownTools = new Set(); // 已展示的工具ID
    this.refreshCount = 0;
  }
  
  getNextRecommendations() {
    this.refreshCount++;
    
    if (this.refreshCount === 1) {
      // 首次访问：使用首页逻辑
      return getHomepageTools();
    } else {
      // 换一换：使用混合逻辑
      return getRefreshedTools(Array.from(this.shownTools));
    }
  }
  
  recordShownTools(tools) {
    tools.forEach(tool => this.shownTools.add(tool.id));
  }
}
```

function trackRefreshAction(refreshCount, currentTools) {
  trackEvent('hot_refresh', {
    refresh_count: refreshCount,
    featured_tools_shown: currentTools.filter(t => t.isFeatured).length,
    non_featured_tools_shown: currentTools.filter(t => !t.isFeatured).length,
    timestamp: Date.now()
  });
}
